# Advanced Calculator App

A feature-rich calculator designed to go beyond basic arithmetic by combining functionality, usability, and a more engaging user experience.

## Features

### Core Functionality

* Supports standard operations:

  * Addition, subtraction, multiplication, and division
* Exponential calculations
* Built-in mathematical constants:

  * Pi (π)
  * Euler’s number (e)

### History Tracking

* Automatically records previous calculations
* Allows users to review past expressions and results
* Useful for multi-step problem solving without losing progress

### Dynamic User Interface

* Color-changing interface for a more interactive experience
* Adds customization and visual feedback beyond a standard calculator

### Additional Features

* Integrated fun facts to make the experience more engaging
* Designed to feel less rigid than a traditional calculator
